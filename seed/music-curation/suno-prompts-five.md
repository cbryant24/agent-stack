## 🎓 Key Learnings About Suno

### Prompt Structure
**Suno uses TWO separate input fields:**

1. **Style of Music** - Defines the sonic characteristics
   - Genre + tempo + instrumentation + mood + production style
   - Vocal characteristics (if using vocals)
   - All in one combined, flowing prompt
   - Keep under 1000 characters

2. **Lyrics** - Defines structure and words
   - Structure tags in [brackets]
   - Actual lyrics or vocal phrases
   - Metatags for control

### Vocal Control Techniques

**Specifying Vocal Character (in Style field):**
- "breathy female vocals" - intimate, close-mic sound
- "soft airy female voice" - ethereal quality
- "ethereal vocal texture" - atmospheric
- "sparse vocal delivery" - tells Suno to use vocals minimally
- Can specify: "raspy", "smooth", "warm", "nasal", "velvety"

**Controlling Vocal Placement (in Lyrics field):**
- `[Instrumental]` - No vocals, pure beat
- `[Vocal Hook]` - Brief vocal moments
- `[Brief Vocals]` - Very short phrases
- `(parentheses)` - Ad-libs like (ooh, ooh) or (hey!)

### Essential Structure Tags

**Song Sections:**
- `[Intro]` - Beginning (can be unreliable, try `[Instrumental Intro]`)
- `[Verse]` - Story/content sections
- `[Pre-Chorus]` - Build tension
- `[Chorus]` - Main hook
- `[Bridge]` - Contrast section
- `[Outro]` - Ending
- `[Break]` - Pause in lead melody
- `[Hook]` - Catchy repeated part
- `[Fade Out]` - Gradual volume decrease

**Instrumental Controls:**
- `[Instrumental Intro]`
- `[Instrumental Break]`
- `[Instrumental Bridge]`
- `[Instrumental Outro]`
- `[Guitar Solo]`, `[Bass Drop]`, etc.

### Best Practices for Sporadic Vocals

1. **In Style Prompt:**
   - Specify vocal character ("breathy female vocals")
   - Add "sparse vocal delivery" or "minimal vocals"
   - Include language if not English ("French female vocals", "Japanese female vocals")

2. **In Lyrics Field:**
   - Use mostly `[Instrumental]` sections
   - Add `[Vocal Hook]` or `[Brief Vocals]` sparingly (2-4 times total)
   - Keep phrases SHORT: 2-4 words maximum
   - Repeat phrases with (parentheses) for emphasis
   - Use (ad-libs) like (ooh, ooh), (ah, ah), (mm, mm)

3. **Structure Pattern for Sporadic Vocals:**
```
[Instrumental Intro]
[Instrumental Verse]
[Vocal Hook] - 2-4 words
[Instrumental Break]
[Instrumental Verse]
[Vocal Hook] - 2-4 words
[Instrumental Bridge]
[Brief Vocals] - 2-4 words
[Instrumental Outro]
```

### What Worked vs. Didn't Work

**✅ What User Loved:**
- Hardcore phonk with heavy bass focus
- Removal of melodic string instruments for rawer sound
- Memphis cowbell inclusion
- Slowed + reverb versions
- Lo-fi fusion with phonk
- French vocals over lo-fi phonk beats

**❌ What Didn't Work:**
- Metro Boomin cinematic trap style (too polished/orchestral)
- Asian string instruments (koto, shamisen) - too melodic
- Overly dramatic/theatrical production

**🎯 Success Formula:**
```
Underground aesthetic + Heavy bass + Memphis phonk elements + 
Lo-fi textures + Minimal sporadic vocals + Raw/gritty production
```

### Iteration Process
The successful prompts emerged through:
1. Initial broad concept → Too polished
2. Focus on specific artist (Ryan Celsius) → Getting closer
3. Remove melodic elements, add aggression → **Hit the target**
4. Create variations (slowed, lo-fi) → All successful
5. Add multilingual vocals → French version was the winner

---

## 🎯 Final Recommendations

**For Future Hip-Hop/Phonk Production:**
- Start with Ryan Celsius "TRAPPIN IN JAPAN" aesthetic as baseline
- Always include Memphis phonk cowbell
- Emphasize bass weight and distortion
- Keep it raw/underground rather than polished
- Lo-fi textures complement phonk perfectly
- French vocals particularly effective for this style
- Structure sporadic vocals: 80% instrumental, 20% brief vocal hooks

**Prompt Length:**
- Aim for 800-1000 characters in Style field
- Keep lyrics minimal for sporadic vocal effect
- Total word count for vocals: 10-20 words maximum across entire track
