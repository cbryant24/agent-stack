## Key Learnings

### BPM as Mood Control

| BPM Range | Mood Effect |
|-----------|-------------|
| 65-70 | Deep sadness, profound stillness, grief |
| 70-78 | Melancholic, contemplative, slow reflection |
| 80-90 | Balanced, chill, relaxed but present |
| 100-110 | Cheerful, cozy, light energy |
| 120-130 | Upbeat, busy, productive, energetic |

### Key Signature Impact
- **Major key** → Cheerful, wholesome, optimistic, cozy
- **Minor key** → Sad, melancholic, introspective, heavy
- **Seventh chords** → Jazz sophistication, warmth, complexity
- **Suspended/unresolved harmony** → Tension, longing, incompleteness

### Texture & Production Descriptors That Work

**For Lo-Fi Warmth:**
- Vinyl crackle, tape saturation, tape wobble
- Dusty, hazy, warm analog
- Muted, soft, detuned

**For Spaciousness:**
- Spacious arrangement, breathing room between notes
- Sparse, minimal, distant

**For Cozy/Clean:**
- Clean warm production, soft, gentle
- Wholesome, tender, cozy

### Atmosphere Keywords by Time of Day

| Time | Keywords |
|------|----------|
| Morning | Morning mist, sunrise, golden hour, peaceful awakening |
| Afternoon | Sunny, warm, golden afternoon light, summer |
| Evening | Sunset, golden hour, rooftop vibes, dusk |
| Night | Nocturnal, late-night, 3am, neon, urban solitude |
| Deep Night | 4am, sleeping village, midnight tranquility, stars |

### Genre Fusion Patterns That Work

1. **Lo-fi + Orchestral** → Add "string section swells," "cello undertones," "French horn accents"
2. **Lo-fi + Eastern** → Add "koto," "shamisen," "bamboo flute," "pentatonic," "temple bell"
3. **Lo-fi + Trip-Hop** → Add "downtempo breakbeat," "noir atmosphere," "ethereal vocal chops"
4. **Lo-fi + Acoustic** → Add "fingerpicked nylon guitar," "upright bass," "bossa nova influence"
5. **Lo-fi + Cosmic** → Add "celestial pad textures," "floating weightlessness," "interstellar"

### Prompt Structure That Works

```
[Genre/Style], [Sub-genre], [BPM], [Primary instruments], [Secondary instruments], 
[Production texture], [Key/harmony], [Primary mood], [Atmosphere/setting], 
[Secondary mood descriptors], [Specific vibe details], instrumental
```

### Quick Tips

1. **Always end with "instrumental"** if you don't want vocals
2. **Stack 3-4 mood words** for reinforcement (contemplative, wistful, bittersweet)
3. **Include a setting/scene** (late-night rooftop, rainy window, sleeping village)
4. **Specify drum style** (boom-bap, brushed, muted, soft snares)
5. **Layer textures** (vinyl + tape + haze = maximum warmth)
6. **Use time references** (3am, golden hour, midnight) to set atmosphere
