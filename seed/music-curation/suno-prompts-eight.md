## Key Learnings from Suno

### 1. Never Reference Artists by Name
**Problem:** Suno can't legally replicate specific artists  
**Solution:** Translate artistic qualities into descriptive language
- Instead of "Ray Charles style" → "orchestral soul, warm piano, lush strings"
- Instead of "Skip James style" → "delta blues, haunting falsetto, sparse fingerpicking"

### 2. Combined Prompt Structure
**Always merge everything into one flowing prompt:**
- Genre tags + Tempo + Instrumentation + Production texture + Mood + Atmosphere
- Never separate "Style of Music" from descriptive elements
- Suno's Custom mode "Style of Music" field = complete combined prompt

### 3. Essential Prompt Components
```
[Genre] + [Tempo/BPM] + [Key instruments] + [Production quality] + 
[Mood keywords] + [Atmospheric details] + [Vocal direction or [Instrumental]]
```

### 4. Character Limit
- Stay under **1000 characters** (Suno's limit)
- Prioritize evocative descriptors over redundancy

### 5. Instrumental-First Workflow
1. Generate instrumental versions with `[Instrumental]` tag
2. Find the musical foundation you like
3. Remove tag and add vocal style descriptors for lyrical versions
4. Can let Suno auto-generate lyrics OR paste custom lyrics

### 6. Vocal Style Additions
When ready to add vocals, append to your instrumental prompt:

**Soulful vocals:**
```
smooth baritone vocals, warm emotional delivery, slight rasp, soulful phrasing
```

**Delta blues vocals:**
```
high falsetto vocals, haunting delivery, raw emotional intensity, gospel-tinged phrasing
```

### 7. Mood Vocabulary That Works
**Atmospheric descriptors:**
- Contemplative, introspective, melancholic, haunting, eerie
- Smoky, intimate, dusty, vintage warmth
- Bittersweet longing, emotional depth, raw intensity

**Production texture descriptors:**
- Analog tape saturation, vintage recording warmth
- Field recording quality, recording crackle
- Dusty pre-war aesthetic, vintage warmth meets modern clarity
