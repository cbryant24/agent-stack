# Environment & tooling

- macOS, iTerm2 + zsh. Also work on Raspberry Pi.
- Prefer terminal-based solutions over GUI. Provide a script when it's more reusable than one-off commands.

- bash for simple shell tasks.
- Python for logic, data processing, parsing, or error handling.
